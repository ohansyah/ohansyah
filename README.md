# About Me:
Hello World!<br>I help businesses build and stabilize backend systems that handle orders, payments, inventory, and reporting.<br>

My background is backend-heavy Laravel development for Cashier / POS systems, Online Stores, 3rd Party Integration and internal business tools, where reliability and data integrity matter more than flashy UI.<br>

### What I typically work on:
- POS / cashier systems (transactions, stock, daily reports)
- E-commerce backends (orders, checkout, admin panels)
- REST APIs (JWT / token auth, session handling)
- Payment and workflow integrations
<br>

### How I work:
- Clear API and database design
- Defensive backend logic (validation, transactions, rollbacks)
- Practical architecture wothout over engineering
- Focus on business flows, not just endpoints
<br>

### Primary Tech stack:
PHP, Laravel, MySQL/MariaDB, Redis, GIT, REST API (I use tools to solve problems)


### Additional Stack:
GO, Nodejs, Bun, Python, Tailwind, Livewire, Elasticsearch, AWS (EC2, ECS, SNS, SQS, Lambda Etc), CI/CD, Docker, Jenkins.
